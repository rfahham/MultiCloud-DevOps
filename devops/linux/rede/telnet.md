# TELNET

Utilizado para verificar comunicação entre servidores

## Instalar

Ubuntu/Centos

```bash
sudo apt install -y telnet
```

Fedora 

```bash
sudo yum install -y telnet
```

## Verificar local de instalação

```bash
which telnet
/usr/bin/telnet
```

## Verificando comunicação

```bash
telnet [host ou ip] [porta]
```
