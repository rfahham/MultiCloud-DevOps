# Funções

Utilizados para empacotar determinadas tarefas

- Soma
- Subtração
- Multiplicação
- Divisão

Construindo uma função para somar dois números

```bash
def soma(a, b): 
    total = a + b
    print(total)

soma(5, 10)
soma(10, 10)
```

Construindo uma função para subtrair dois números

```bash
def soma(a, b): 
    total = a - b
    print(total)

soma(10, 5)
soma(20, 10)
```

Construindo uma função para multiplicar dois números

```bash
def soma(a, b): 
    total = a * b
    print(total)

soma(10, 5)
soma(20, 10)
```
