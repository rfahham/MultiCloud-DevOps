# Security Group

## Criando

<div align="center">

![Criar EC2](../images/criar-ec2.png)

</div>

Nomear o Security Group

Acrescentar uma Descrição par ao Security Group

Tanto na Regra de Entrada como a de Saída, adicionar o IP da máquina local.

Pode ser obtido pelo site: https://www.whatismyip.com/

Depois, Clicar em Criar Grupo de Segurança.

>Importante

Setando seu IP, evita o acesso externo ao seu ambiente
